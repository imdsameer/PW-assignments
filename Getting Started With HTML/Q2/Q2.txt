Q2) Explain the purpose of comments in HTML and provide an example of how to use comments in an HTML
documents.

answer: 

Purpose of Comment tags in HTML-->

*)This tag is used to explain the code 
and also sometimes used as a checkpoint in the code for further editing and debugging.
*)This tag is used to insert self understanding language in the code which is not rendered by the browser, i.e the text inside the comment tag is ignored by the browser
*)This tag is used to increases code readability by helping us and others to understand our code.



Example :-
                    <!--These texts are not rendered by the browser-->
                 <h1>Full Stack web development </h1>     <!--this is our heading-->
                <p> This is a paragraph </p>               <!--this is first paragraph-->
