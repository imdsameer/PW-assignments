HTML Tags

Ordered lists

  1. < button > Defines a clickable button
  2. < body > Defines the document's body
  3. < br > Defines a single line break
  4. < img > Defines an image
  5. < input > Defines an input control
  6. < small > Defines smaller text
  7. < form > Defines an HTML form for user input
  8. < footer > Defines a footer for a document or section
  9. < table > Defines a table
  10. < tr > Defines a row in a table
  11. < u > Defines some text that is unarticulated and styled differently from normal text
  12. < ul > Defines an unordered list
  13. < head > Contains metadata/information for the document
  14. < p > Defines a paragraph
  15. < a > Defines a hyperlink
